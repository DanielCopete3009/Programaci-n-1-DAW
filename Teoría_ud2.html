Unidad 2 - Tipos de datos avanzados

Listas (Lists)

Las listas son el "array" principal de Python. Son colecciones ordenadas y mutables (se pueden cambiar). Se definen con corchetes `[]`. 

# Crear una lista
frutas = ["manzana", "banana", "naranja"]

# Acceder a un elemento (índice empieza en 0)
print(frutas[0])  # Output: manzana

# Añadir un elemento al final
frutas.append("uva")

# Recorrer la lista
for f in frutas:
    print(f)

Tuplas (Tuples)

Las tuplas son similares a las listas, pero son inmutables (no se pueden cambiar después de crearlas). Son más rápidas que las listas y se usan para datos que no deben modificarse. Se definen con paréntesis `()`. 

# Crear una tupla
coordenadas = (10.0, 20.5)

# Acceder a un elemento
print(coordenadas[1])  # Output: 20.5

# Desempaquetar una tupla
x, y = coordenadas
print(f"X es {x}, Y es {y}")

# Esto daría un ERROR (son inmutables)
# coordenadas.append(30.0) 
# coordenadas[0] = 5.0

Diccionarios (Dictionaries)

Los diccionarios son colecciones no ordenadas (en versiones antiguas de Python) que almacenan datos en pares de clave-valor. Son extremadamente útiles para organizar datos. Se definen con llaves `{}`. 

# Crear un diccionario
alumno = {
    "nombre": "Daniel",
    "edad": 20,
    "curso": "DAW"
}

# Acceder a un valor usando su clave
print(alumno["nombre"])  # Output: Daniel

# Añadir un nuevo par clave-valor
alumno["nota"] = 9.5
print(alumno)

# Recorrer un diccionario
for clave, valor in alumno.items():
    print(f"{clave}: {valor}")
