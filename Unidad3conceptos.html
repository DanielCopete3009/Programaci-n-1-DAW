Unidad 3 - Fundamentos de Java

I. Introducción a Java

[cite_start]Java es un lenguaje de programación orientado a objetos y plataforma independiente[cite: 101, 102]. Esto significa que los programas Java pueden ejecutarse en cualquier sistema con la Máquina Virtual de Java (JVM).[cite: 103].

[cite_start]Su sintaxis está basada en C/C++, pero simplificada y más segura[cite: 105].

II. Variables y Tipos de Datos

[cite_start]Las variables almacenan datos y deben declararse con un tipo específico[cite: 115].

    [cite_start]
  • Tipos primitivos: `int`, `double`, `float`, `char`, `boolean`[cite: 116].
    • [cite_start]
  • Tipos referencia: Objetos como `String`, arrays y clases personalizadas[cite: 118].

[cite_start]Ejemplo de declaración e inicialización:[cite: 120] 

// Variable entera
int edad = 20;

// Variable decimal
double precio = 99.99;

// Cadena de texto
String nombre = "Daniel";

Constantes

[cite_start]

Se declaran con `final` y su valor no puede cambiar:

final double PI = 3.1416;

IV. Estructuras de Control

Condicionales

int numero = 10;

if (numero > 0) {
    [cite_start]System.out.println("Positivo"); [cite: 165]
} else if (numero < 0) {
    System.out.println("Negativo");
} else {
    System.out.println("Cero");
}

Bucles

// Bucle for
for (int i = 0; i < 5; i++) {
    [cite_start]System.out.println(i); [cite: 172]
}

// Bucle while
int j = 0;
while (j < 5) {
    System.out.println(j);
    j++;
}

// Bucle do-while
int k = 0;
do {
    System.out.println(k);
    k++;
} while (k < 5);

V. Arreglos (Arrays)

[cite_start]Los arrays son estructuras de datos que almacenan elementos del mismo tipo[cite: 190]. Se accede a ellos mediante un índice comenzando en 0. 

// Declaración e inicialización
int[] numeros = {1, 2, 3, 4, 5};

// Acceso y modificación
numeros[2] = 10; // cambia el 3 por 10

// Recorrido
for (int i = 0; i < numeros.length; i++) {
    [cite_start]System.out.println(numeros[i]); [cite: 198]
}

VI. Métodos

[cite_start]Los métodos permiten encapsular código que puede reutilizarse y recibir parámetros[cite: 220]. 

public static int suma(int a, int b) {
    [cite_start]return a + b; [cite: 222]
}

// Llamada
int resultado = suma(5, 3);
