Unidad 2 - Tipos de datos avanzados

Teoría

En esta unidad trabajamos con arrays y expresiones regulares (regex).

Arrays

Un array almacena múltiples valores del mismo tipo.

# Array unidimensional en Python
numeros = [10, 20, 30, 40]
for n in numeros:
    print(n)

Expresiones Regulares

Regex permite buscar o validar patrones en cadenas de texto.

# Validación de email en Python
import re

correo = "usuario@ejemplo.com"
if re.match(r"^[\w\.-]+@[\w\.-]+\.\w+$", correo):
    print("Correo válido")
else:
    print("Correo no válido")
