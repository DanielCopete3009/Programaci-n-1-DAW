Unidad 4 - Arrays y Cadenas de Caracteres

I. Introducción a las Estructuras de Datos

[cite_start]Las variables escalares (como `edad = 6`) solo pueden almacenar un valor a la vez[cite: 162, 163, 164].

[cite_start]Las estructuras de datos, como las tablas o arrays, permiten almacenar múltiples valores en una sola variable[cite: 170]. Se clasifican en:

    [cite_start]
  • Arrays (Estáticas): Tienen un tamaño fijo y sus elementos son homogéneos (todos del mismo tipo)[cite: 177].
    • [cite_start]
  • Listas (Dinámicas): Tienen un tamaño variable[cite: 179].

II. Arrays (Listas en Python)

Un array es una estructura que almacena un conjunto de elementos de un mismo tipo. [cite_start]Se accede a ellos mediante un índice[cite: 215].

    [cite_start]
  • El índice siempre comienza en 0[cite: 216].
    • [cite_start]
  • El último índice es N-1 (donde N es el número total de elementos)[cite: 217].

En Python, no existen los arrays tradicionales. [cite_start]En su lugar, se implementan usando listas[cite: 225]. [cite_start]Las listas de Python son mutables (su contenido puede modificarse) y ordenadas[cite: 228, 229].

Declaración e Inicialización

# Lista con elementos
[cite_start]mi_array = [10, 20, 30, 40] [cite: 236]

# Lista vacía
[cite_start]otra_lista = [] [cite: 238]

# Pre-asignación (ej: 5 ceros)
[cite_start]ceros = [0] * 5 [cite: 239]

Acceso y Modificación por Índice

[cite_start]

Se usa el índice (de 0 a N-1) para acceder o cambiar elementos[cite: 245].

mi_array = [22, 8, 97, 3]

# Acceso (primer elemento)
[cite_start]print(mi_array[0]) # Output: 22 [cite: 247]

# Modificación
[cite_start]mi_array[2] = 100 # Cambia el 97 por 100 [cite: 249]

# Acceso Negativo (último elemento)
[cite_start]print(mi_array[-1]) # Output: 3 [cite: 251]

Recorrido: Bucles y Listas

# 1. Recorrido por índice (tradicional)
for i in range(len(mi_array)):
    [cite_start]print(mi_array[i]) [cite: 271, 272]

# 2. Recorrido por elemento (for-each)
for elemento in mi_array:
    [cite_start]print(elemento) [cite: 273, 275]

Operaciones y Métodos Básicos

    [cite_start]
  • Slice (División): Extrae una sub-lista usando `lista[inicio:fin:paso]`[cite: 380]. [cite_start]`fin` no se incluye[cite: 384].
    • [cite_start]
  • `append(x)`: Añade un elemento `x` al final de la lista[cite: 278, 418].
    • [cite_start]
  • `insert(i, x)`: Inserta el elemento `x` en la posición (índice) `i`[cite: 284, 434].
    • [cite_start]
  • `extend(otra_lista)`: Añade todos los elementos de `otra_lista` al final[cite: 453, 461].
  • `remove(x)`: Elimina la primera ocurrencia del valor `x`. [cite_start]No devuelve nada[cite: 289, 477, 478].
  • `pop(i)`: Elimina el elemento en el índice `i` y lo devuelve. [cite_start]Si `i` está vacío, elimina y devuelve el último[cite: 291, 504, 513, 515].

Operaciones Avanzadas

  • Búsqueda (Python):
    • `in` (Operador): Comprueba si un elemento existe. [cite_start]`if "Ana" in lista:`[cite: 648, 650].
    • `.index(x)` (Método): Devuelve el índice de `x`. [cite_start]Da error si no lo encuentra (usar `try...except`)[cite: 648, 655, 657].
  • Ordenación:
      [cite_start]
    • `.sort()`: Modifica la lista original (la ordena "in-place")[cite: 694].
      • [cite_start]
    • `sorted(lista)`: Devuelve una nueva lista ordenada, dejando la original intacta[cite: 695].
  • Comparación (`==`):
    • Compara si dos listas son idénticas. [cite_start]Deben tener la misma longitud y los mismos elementos en el mismo orden[cite: 775, 777, 779].
  • Copia de Arrays:
      [cite_start]
    • Forma Errónea: `lista_b = lista_a` NO es una copia[cite: 844]. Es solo otro nombre (una referencia). [cite_start]Si modificas `lista_b`, también cambias `lista_a`[cite: 844, 868].
    • Forma Correcta: `lista_b = lista_a.copy()` o `lista_b = lista_a[:]` (usando slicing). [cite_start]Esto crea una copia real[cite: 874, 875].

Arrays de Dos Dimensiones (Matrices)

    [cite_start]
  • Concepto: Una tabla con filas y columnas[cite: 912]. [cite_start]En Python, se implementa como listas anidadas (una lista que contiene otras listas)[cite: 913].
    • [cite_start]
  • Creación: `matriz = [ [1, 2, 3], [4, 5, 6], [7, 8, 9] ]`[cite: 918, 921, 922, 924].
    • [cite_start]
  • Acceso: Se usan dos índices: `matriz[fila][columna]`[cite: 936]. (Ej: `matriz[1][2]` sería 6) [cite_start][cite: 938, 959].
  • Recorrido: Se usan dos bucles `for` anidados. [cite_start]El exterior para las filas y el interior para las columnas (elementos)[cite: 975].

III. Cadenas de Caracteres (Strings)

[cite_start]Una cadena es una secuencia de caracteres (letras, números, símbolos)[cite: 1075].

[cite_start]Concepto Clave: Las cadenas son INMUTABLES[cite: 1088]. Una vez creadas, no se puede cambiar un carácter individual. [cite_start]Cualquier "modificación" (como un `.replace()`) crea una cadena completamente nueva[cite: 1089].

[cite_start]Se crean con comillas simples (`' '`), dobles (`" "`) [cite: 1079] [cite_start]o triples (`""" """`) para multilínea[cite: 1082].

Indexación y Slicing

Funciona igual que en las listas.

    [cite_start]
  • Índice: `cadena[0]` (primera letra) [cite: 1097][cite_start], `cadena[-1]` (última letra)[cite: 1098].
    • [cite_start]
  • Slicing: `cadena[inicio:fin:paso]`[cite: 1115]. (Ej: `cadena[0:5]` extrae los primeros 5 caracteres) [cite_start][cite: 1126].
    • [cite_start]
  • Invertir: `cadena[::-1]`[cite: 1139].

Operaciones Básicas

  • Concatenación (`+`): Une cadenas. [cite_start]`"Hola" + " " + "Mundo"`[cite: 1143].
  • Repetición (`*`): Repite la cadena. [cite_start]`"-" * 10` (crea 10 guiones)[cite: 1152].
    • [cite_start]
  • Longitud (`len(cadena)`): Devuelve el número de caracteres[cite: 1155].

Métodos Principales

    [cite_start]
  • `.lower()` / `.upper()`: Convierten a minúsculas o mayúsculas[cite: 1161].
    • [cite_start]
  • `.strip()`: Elimina espacios en blanco al inicio y al final[cite: 1161].
    • [cite_start]
  • `.replace(a, b)`: Reemplaza todas las `a` por `b`[cite: 1161].
    • [cite_start]
  • `.split(separador)`: Divide la cadena en una lista[cite: 1161]. (Ej: `"a,b,c".split(',')[cite_start]` da `['a', 'b', 'c']`)[cite: 1161].
    • [cite_start]
  • `.join(iterable)`: Une los elementos de una lista en una sola cadena[cite: 1161]. (Ej: `"-".join(['a', 'b'])[cite_start]` da `"a-b"`)[cite: 1161].
    • [cite_start]
  • `.find(sub)`: Devuelve el índice de la subcadena `sub` (o -1 si no la encuentra)[cite: 1161].
    • [cite_start]
  • `.startswith(sub)`: Devuelve `True` si la cadena empieza con `sub`[cite: 1161].

Formato de Cadenas (f-strings)

[cite_start]Es la forma moderna y recomendada[cite: 1167]. [cite_start]Se usa una `f` antes de las comillas y las variables se ponen entre llaves `{}`[cite: 1168]. 

producto = "Laptop"
precio = 850.50
[cite_start]mensaje = f"El {producto} cuesta ${precio:.2f}." [cite: 1169, 1170]
# [cite_start]Salida: El Laptop cuesta $850.50. [cite: 1171]

IV. Expresiones Regulares (RegEx)

[cite_start]Las Expresiones Regulares (o RegEx) son patrones de búsqueda[cite: 1246]. [cite_start]Son un mini-lenguaje universal para buscar y manipular texto[cite: 1246].

[cite_start]

En Python, se usa el módulo `re`[cite: 1247]. [cite_start]Se recomienda usar "raw strings" (con una `r` antes de las comillas) para los patrones[cite: 1302].

Funciones Principales del Módulo `re`

    [cite_start]
  • `re.search(patron, texto)`: Busca el patrón en cualquier parte del texto[cite: 1272].
    • [cite_start]
  • `re.findall(patron, texto)`: Encuentra todas las coincidencias y las devuelve como una lista[cite: 1274].
    • [cite_start]
  • `re.match(patron, texto)`: Solo busca si el patrón coincide al inicio del texto[cite: 1275].

Patrones Básicos (Comodines)

    [cite_start]
  • `.` (Punto): Cualquier carácter (excepto salto de línea)[cite: 1281].
    • [cite_start]
  • `\d` (Dígito): Cualquier dígito del 0 al 9[cite: 1281].
    • [cite_start]
  • `\w` (Word): Cualquier carácter de palabra (letras, números, guion bajo)[cite: 1281].
    • [cite_start]
  • `\s` (Space): Cualquier espacio en blanco (espacio, tabulador, etc.)[cite: 1281].
    • [cite_start]
  • `\D`, `\W`, `\S` (Mayúsculas): Son la negación de los anteriores (No-Dígito, No-Palabra, No-Espacio)[cite: 1282].

Cuantificadores (¿Cuántas veces?)

    [cite_start]
  • `*` (Asterisco): Cero o más veces[cite: 1285].
    • [cite_start]
  • `+` (Más): Una o más veces[cite: 1285].
    • [cite_start]
  • `?` (Interrogación): Cero o una vez (hace que sea opcional)[cite: 1285].
  • `{n}` (Llaves): Exactamente `n` veces. (Ej: `\d{4}` busca 4 dígitos) [cite_start][cite: 1285, 1293].
    • [cite_start]
  • `{n,m}` (Llaves): Un rango, entre `n` y `m` veces[cite: 1285].
